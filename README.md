### Installation  

# clone the repo

git clone https://github.com/MukeshKumarVishwakarma/ShopNest-Shopping.git
cd ShopNest-Shopping

# install dependencies
npm install

# Running Locally
npm run dev

# Building for Production
npm run build

# Previewing Production Build
npm run serve
